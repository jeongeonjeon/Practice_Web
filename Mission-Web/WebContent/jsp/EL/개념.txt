#EL(Expression language)
 -> 표현식<%= %>을 대신하기 위해 사용
 -> 자바코드가 아닌 jsp코드를 사용하기 위해 만들어짐
 -> jsp에서 제공하는 내장객체 중 4가지 기본 영역에 대한 속성 사용
 -> 사용법 : ${expr}
 -> EL표기법이 좋은 이유/선호하는 이유 : null문자가 아닌 ""공백으로 출력된다
 
 #내장객체
  - pageContext
  - pageScope
  - requestScope
  - sessionScope
  - applicationScope
  - param : 서블릿의 request.getParameter()와 같음
  - paramValues : 서블릿의 request.getParameterValues()와 같음
  - cookie
  
# 공유영역 : jsp와 El사이의 객체공유
 - jsp의 공유영역 : pageContext,  request  ,    session   ,   application
 - EL의  공유영역   : pageScope,  requestScope, sessionScope, applicationScope
 -  공유영역의 범위 :  좁음 <------------------------------------------->넓음
	 (좁은 범위부터 넓은 범위 순으로 찾는다.)
	 

# EL에서의 객체 표현 이해
 - ${객체.이름}	(이름 : getter메소드를 의미)
 	-> 객체 자리에 올 수 있는 타입 2가지 : 자바빈즈 클래스와 맵 객체만 가능
 		※자바빈즈 클래스 : VO클래스
 			-패키지로 선언(default package불가)
 			-클래스 public선언
 			-매개변수 없는 생성자 선언
 			-멤버변수는 private선언
 			-멤버변수에 대한 setter와 getter 메소드 선언
 	
 - ${객체[이름]}	(이름:collection 객체)
